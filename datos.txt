1. Nmap — Reconocimiento
Escaneamos la máquina para ver qué puertos estaban abiertos. Encontramos SSH (22) y HTTP (80). Esto nos dice cómo podemos interactuar con la máquina.
2. Gobuster — Enumeración web
Buscamos directorios ocultos en la web. No encontramos nada útil, así que descartamos esa vía.
3. Hydra — Fuerza bruta SSH
Probamos miles de combinaciones de usuario/contraseña automáticamente contra el SSH. Encontramos mario:chocolate. La clave fue usar un usuario específico (mario) en vez de una lista enorme.
4. SSH — Acceso inicial
Con las credenciales entramos a la máquina como usuario normal mario.
5. sudo -l — Buscar privilegios
Este comando muestra qué puede ejecutar tu usuario como root. Descubrimos que mario podía ejecutar vim como root.
6. Vim — Escalada de privilegios
Vim permite ejecutar comandos del sistema desde dentro. Como lo ejecutamos como root, pudimos lanzar una bash también como root. Esto se llama GTFOBins — binarios legítimos que pueden usarse para escalar privilegios.
